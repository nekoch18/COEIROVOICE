# COEIROVOICE
CEIROINKのAPIを使って音声を出せる簡単なデスクトップアプリです。
## 機能
* 直感的なUI: テキストを入力してボタンを押すだけの簡単操作。
* キャラクター切り替え: COEIROINKに登録されているキャラクターを動的に取得・選択可能
## 🛠 事前準備
本アプリを使用するには、COEIROINK が起動している必要があります、また音声を再生するためにffmpeg(ffplay)、音声をマイク音声に載せるためにSYNCROOMをインストールしてください。(仮想マイク音声に出来れば何でもいい)
* [COEIROINK 公式サイト](https://coeiroink.com)からCOEIROINKをダウンロード。
* [FFmpeg 公式サイト](https://ffmpeg.org)からFFmpegをダウンロード&インストール。
* [SYNCROOM 公式サイト](https://syncroom.yamaha.com)からSYNCROOMをダウンロード&インストール。SYNCROOMを推奨）
## 使用
1. [Releases](https://github.com/nekoch18/COEIROVOICE/releases) から最新の COEIROVOICE-{Version}.exe をダウンロードします。
2. COEIROVOICE-{Version}.exe を実行してください。
3. 初回使用時にCOEIROINKの実行EXEファイルをしてください。
4. COEIROVOICEの上のタブからCOEIROVOICE起動し、テキストを入力し、再生を行います。
